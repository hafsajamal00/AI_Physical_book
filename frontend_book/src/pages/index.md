# Welcome to the ROS 2 Educational Book

This comprehensive guide will help you understand ROS 2 as the middleware that connects AI agents (digital brain) to humanoid robot hardware and simulation.

## What You'll Learn

In this educational module, you will:

- Understand the fundamentals of ROS 2 and its importance for Physical AI
- Learn about ROS 2 architecture and the Data Distribution Service (DDS)
- Explore the differences between using ROS 2 in simulation versus with real robots
- Master the ROS 2 communication model including nodes, topics, and services
- Understand message passing and coordination mechanisms
- Learn Python integration using rclpy
- Explore robot structure and control basics
- Work with URDF for humanoid robots
- Learn how to bridge Python AI agents to ROS controllers

## Getting Started

To begin your journey into the world of ROS 2 and Physical AI, start with the [Introduction](./docs/intro) or jump directly to the tutorials in the sidebar.
